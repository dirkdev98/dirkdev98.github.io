First post

Hello! Time to get a personal blog going. Build a quick setup to statically convert Markdown pages to HTML including a dev server for previews.

The stack

Using front-matter to extract page titles, descriptions and the actual markdown from files. Then it is passed through marked to convert Markdown to HTML. The dev server uses a Compas provide Koa instance and renders the pages on the fly.

Extra's

On top of converting the pages to HTML it also adds some breadcrumbs and 'index'-navigation pages, to make it easier to navigate through the site when landing on an article.
